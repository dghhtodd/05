# 萌乐游戏摄影棚

一个多语言支持的游戏摄影展示网站，提供简体中文、繁体中文和英文三种语言版本。

## 功能特点

- 多语言切换（简体中文、繁体中文、英文）
- 响应式设计，支持移动端和桌面端
- 社交媒体链接整合（TikTok、抖音、Instagram等）
- 优雅的UI设计和动画效果

## 项目结构

```
.
├── index.html          # 主页面
├── js/
│   └── lang.js        # 多语言支持实现
└── assets/
    └── placeholder.jpg # 占位图片
```
